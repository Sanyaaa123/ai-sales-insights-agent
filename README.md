# AI Sales Insights Agent

An AI-powered sales analytics dashboard that automatically generates KPIs, visualizations, and business insights from uploaded datasets.

## Features

• Upload any sales dataset
• Automatic KPI metrics generation
• Interactive sales analytics dashboard
• AI-generated business insights
• Natural language data analysis using AI agent

## Tech Stack

Python
Pandas
Streamlit
Plotly
OpenAI API

## Example Insights

* Revenue by region analysis
* Customer segmentation insights
* Product profitability analysis
* Monthly sales trends

## How to Run

```bash
pip install -r requirements.txt
streamlit run app/streamlit_app.py
```

## Project Structure

```
app/streamlit_app.py → Main dashboard  
pipeline/ → Data processing  
data/ → Sample dataset
```
